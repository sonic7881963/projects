# SJ2215 Text Life Mini Game

An asset and UI-flow folder for a text-life mini game. The project includes preview images, luck-selection assets, initial-attribute assets, and life-event screen resources.

## What This Demonstrates

- Organizing visual assets around a game flow.
- Designing a content-driven interaction structure.
- Preparing UI states for a lightweight narrative/life-simulation game.
- Separating preview, selection, attribute, and life-event resources.

## Folder Structure

| Folder | Purpose |
| --- | --- |
| `0-预览图/` | Preview screenshots for the mini game. |
| `1-选择运气/` | Luck-selection UI assets. |
| `2-初始属性/` | Initial attribute UI assets. |
| `3-开启人生/` | Life-event/start-game UI assets. |
| `bj.png` | Shared background/image resource. |

## Notes

This folder is primarily an asset package rather than a full source-code project. It is useful for reviewing UI state planning, resource naming, and how a simple game flow can be broken into screens and reusable visual elements.
