# QuanMinZhuGong

A Cocos Creator 2.4.6 mini-game project with TypeScript gameplay scripts, combat scenes, prefabs, and image assets. The project is organized around level-specific command scripts and scene resources under `assets/3.16小游戏`.

## What This Demonstrates

- Cocos Creator project organization.
- TypeScript scripting for interactive game scenes.
- Level-specific gameplay logic and enemy behavior.
- Asset integration with prefabs, scenes, and visual resources.

## Tech Stack

Cocos Creator 2.4.6, TypeScript, JavaScript project configuration, Cocos scene and prefab assets.

## Key Files And Folders

| Path | Purpose |
| --- | --- |
| `project.json` | Cocos Creator project metadata. |
| `assets/3.16小游戏/` | Main game assets, scenes, prefabs, and scripts. |
| `assets/3.16小游戏/command_TypeScript/` | Level-specific TypeScript command scripts. |
| `assets/3.16小游戏/*.fire` | Cocos scene files. |
| `assets/3.16小游戏/**/*.prefab` | Reusable game objects and characters. |

## How To Open

Open the folder in Cocos Creator 2.4.6 or a compatible Cocos Creator 2.x editor. The project contains editor-generated metadata files that should be kept with their matching assets.

## Notes

This project is best reviewed as a game-engine workspace. The most relevant engineering artifacts are the TypeScript scripts that coordinate level behavior, state, enemy interactions, and scene transitions.
