# TowerDefense

A legacy tower-defense project folder. At the moment this directory only contains an older placeholder `readMe.txt`; the active Cocos Creator tower-defense implementation in this repository is represented by the `tower` folder.

## What This Folder Represents

- A preserved project slot for tower-defense work.
- A pointer for reviewers who are browsing the repository by project name.
- A lightweight index that keeps the folder understandable even though no source files are currently present here.

## Related Project

See `../tower` for the Cocos Creator 3.4.0 tower-defense style game implementation with TypeScript scripts, prefabs, animations, collision logic, and scene assets.

## Notes

No runnable source code is currently stored in this folder. This README is intentionally factual so reviewers do not mistake the placeholder for a complete implementation.
